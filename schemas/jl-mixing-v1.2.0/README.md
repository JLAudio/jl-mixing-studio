# JL Mixing Automation schemas

These are unchanged copies of the JSON Schemas released by [JL Mixing Automation v1.2.0](https://github.com/JLAudio/jl-mixing/tree/v1.2.0/schemas):

- `studio.schema.json`
- `client.schema.json`
- `project-manifest.schema.json`
- `delivery-manifest.schema.json`

JL Mixing Automation v1.2.0 uses document schema version `1.1.0`. Application release versions and document schema versions are intentionally independent.

The schemas are copied into JL Mixing Studio so workspace validation remains local, deterministic, and available offline. They retain the Apache-2.0 licensing of the source repository.
